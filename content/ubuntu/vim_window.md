vim多文件多窗口操作
----------------

1. 同时打开多个文件 vim fl1 fl2

2. 如果vim已经启动，在当前窗口中再打开一个文件: ```:open file```

3. 文件间切换

  * Ctrl+6—下一个文件
  * :bn—下一个文件
  * :bp—上一个文件

3. 同时显示多个文件（窗格形式）: ```:split ```  ``` :vsplit```

4. 多个窗格之间之间切换: ``` ctrl+w+上下左右方向键/hjkl ```

5. 依次向后切换到下一个窗格: ``` ctrl+ww ```

4. 不关闭文件，临时切换回命令行:shell，再切换回来 exit
